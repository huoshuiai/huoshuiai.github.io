# 黄若兰纪念网站

这是一个献给黄若兰的纪念网站。网站旨在系统展示若兰的生平、作品和故事，让她的美好永存于心。

## 🌟 网站结构

- 首页：若兰的简介和精选照片
- 生平：按时间顺序展示的生平事迹
- 作品：产品设计、报告和文章展示
- 我与若兰：朋友们的回忆
- 若兰奖学金：专门页面

## 🛠️ 技术架构

本站使用 Astro 构建，采用简约、优雅的设计风格。

## 🧞 开发命令

| 命令                     | 作用                                           |
| :----------------------- | :--------------------------------------------- |
| `npm install`            | 安装依赖                                       |
| `npm run dev`            | 启动本地开发服务器，地址：`localhost:4321`     |
| `npm run build`          | 构建生产版本到 `./dist/`                      |
| `npm run preview`        | 本地预览生产版本                               |
